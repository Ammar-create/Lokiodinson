01 — The Merge series

Models fused, not trained

We compose frontier capability by merging specialized parent models — weighing them layer by layer, welding them into one set of weights, then tempering the result against evaluation suites until the weak candidates break. Hours of compute instead of months of training, with a full composition record for every release.

02 — Custom harnesses

Containment, built to fit

An agent is only as reliable as the structure holding it. We design and build bespoke harnesses — tool contracts, budgets, checkpoints, eval hooks, human gates — mapped to your workload's actual failure modes, not a generic framework's guess at them. Works with Merge models or any model you already run.

Know your enemy

Failure modes we build against

Every harness starts from a failure-mode inventory. These four show up in almost every agent deployment we audit.

×Silent loops

The agent retries a broken step forever, quietly burning budget and returning nothing.

Bounded step & spend budgets, loop detection

×Tool drift

An upstream API changes shape. The agent keeps calling it, misreading garbage as signal.

Typed tool contracts, drift halts

×Context rot

Long-running tasks degrade as state accumulates until the agent forgets its own goal.

Checkpoints, state compaction

×Confident nonsense

The model asserts a wrong answer with perfect fluency, then acts on it irreversibly.

Eval hooks, human gates on irreversible actions
